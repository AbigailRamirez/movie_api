MyFlix API DOCUMENTATION

A REST API that interacts with a database that stores data about different movies.

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Get list of all movies to the user /movies GET None A JSON object holding information about all movies.
Get data (description, genre, director, image URL) about a single movie by title. /movies/:title GET None A JSON object holding information about a single movie.
Example:
{
"Title": "Ant-Man And The Wasp - Quantumania",
"Description": "Ant-Man and the Wasp find themselves exploring the Quantum Realm, interacting with strange new creatures and embarking on an adventure that pushes them beyond the limits of what they thought was possible.",
"Genre": { "Name": "Action",
"Description":"Action film is a film genre in which the protagonist is thrust into a series of events that typically involve violence and physical feats.",
},
"Director": {
"Name": "Peyton Reed",
"Bio": "Peyton Tucker Reed is an American television and film director. He directed the comedy films Bring It On, Down with Love, The Break-Up, and Yes Man, as well as the superhero film Ant-Man and its sequels.",
"Birth":1964
},
"ImageURL": "https://www.hollywoodreporter.com/wp-content/uploads/2015/10/ap_935263600928.jpg?w=3000",
"Featured":true
}
Return data about a genre (description) by name/title /movies/genres/:genreName GET None A JSON object holding information about the requested genre.
As such:
{
"Name": "Action",
"Description":"Action film is a film genre in which the protagonist is thrust into a series of events that typically involve violence and physical feats.",
}
Return data about a director (bio, birth year, etc.) by name. /movies/directors/:directorName GET None A JSON object holding information about the requested director.
Example:
{
"Name": "Peyton Reed",
"Bio": "Peyton Tucker Reed is an American television and film director. He directed the comedy films Bring It On, Down with Love, The Break-Up, and Yes Man, as well as the superhero film Ant-Man and its sequels.",
"Birth":1964
}
Allow a new user to register. /users POST A JSON object holding data about the user to add.
Example:
{
"username": "username1",
"movieList": []
} 		A JSON object contaitning data about the new user. Including an id.
Example:
{
"id": 1,
"username": "username1",
"favoriteMovies": []
}
Allow users to update their information /users/:id PUT :id A JSON object containing the updated user information.
Allow user to add a movie to their favorites /users/:id/:movieTitle POST :id/:movieTitle A text message indicating that the movie has successfully been added to the users favorite movies list.
Allow users to remove a movie from their favourites. /users/:id/:movieTitle DELETE :id A text message indicating that the movie has successfully been removed from the favorite movies list
Allow existing users to deregister /users/:id DELETE :id A text message indicating that the user has successfully been removed/deregistered
